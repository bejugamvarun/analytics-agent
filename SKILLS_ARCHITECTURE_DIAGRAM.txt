QUICK START:

1. Install ADK 1.25.0+:
   $ pip install --upgrade google-adk

2. Verify Skills load correctly:
   $ python skills/test_skills.py

3. Run your agent:
   $ adk run src/risk_analytics_agent/orchestrator/agent.py

4. Try a query:
   "Calculate MLO day-over-day variance for March 15th at entity level"
